The red outlines and red text show the locations that the system will use for mapping. If we don't have an area for a location, you'll just see a red dot.

Changing a Location

  1. Click on the red boundary or the red dot.
  2. The Location Name box will update.
  3. Drag the nodes in the boundary, or use the polygon icon (next to the hand) to draw a new one.
  4. Click Save Changes.

Deleting a Location

  1. Click on the red boundary or the red dot.
  2. The Location Name box will update.
  3. Click Delete Location.
