These are recent messages which we think might be spam. They can either be Pending messages which haven't reached your group, or Approved messages which are already on the group.

We err on the safe side, so sometimes you'll see messages which aren't spam here, for you to check.

This is about the specific message, not the sender. If you think this message is spam, click the Spam button and we'll deal with them. If they're not spam, click the Not Spam button and they'll be returned to Pending or Approved as appropriate. If you think the sender is a deliberate spammer, use the Spammer button to report them as well.
